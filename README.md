# Prapti Das — Portfolio

A complete, single-page-app portfolio website. Four pages (Home, Resume, Portfolio, Blog) in one HTML file, with separate CSS and JS. No build step, no framework, no dependencies beyond Google Fonts.

## File Structure

```
prapti-portfolio/
├── index.html          ← Entire site (SPA, all 4 pages)
├── styles.css          ← All styles
├── scripts.js          ← All JS (routing, carousel, snitch, clock, feathers)
├── favicon.svg         ← Tab icon (gothic PD monogram + candle flame)
├── README.md           ← This file
└── assets/
    ├── carousel1.jpg         ← Smash Burger "Everybody Smashes"
    ├── carousel2.jpg         ← Smash Burger "One Bite"
    ├── carousel3.jpg         ← Smash Burger "When The Burger Hits Different"
    ├── zariya-poster.jpg     ← Zariya Productions street play poster
    ├── event_reel.mp4        ← Video: Event Reel
    ├── fast_paced_edit.mp4   ← Video: Fast-Paced Edit
    ├── storyboard.mp4        ← Video: Storyboard
    ├── storytelling.mp4      ← Video: Storytelling
    ├── typography_based.mp4  ← Video: Typography-Based
    └── montage.mp4           ← Video: Montage
```

## Deploy to Vercel (Free)

1. Push this entire folder to a GitHub repository (public or private).
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub.
3. Click **Add New Project** and import the repository.
4. Vercel auto-detects it as a static site. Do not change any settings.
5. Click **Deploy**. Live in ~30 seconds at `your-repo.vercel.app`.

For a custom domain: Project Settings → Domains.
